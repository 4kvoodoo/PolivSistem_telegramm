# Система полива на ESP8266
Для связи микроконтроллера ESP8266 с клиентом используется Telegram-бот.
Основная цель проекта — осуществить удобное управление поливом растений при наименьшем бюджете.

# Подключение к Wi-Fi
При первом включении микроконтроллер нужно подключить к Интернету. В будущем он будет подключаться к нему сам, если, конечно, найдет поблизости ранее заданную точку доступа Wi-Fi.
